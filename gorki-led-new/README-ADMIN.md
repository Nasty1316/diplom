# Инструкция по использованию админ панели

## 🚪 Вход в админ панель

1. Откройте файл `admin/login.html` в браузере
2. Введите данные для входа:
   - **Логин:** `admin`
   - **Пароль:** `admin123`
3. Нажмите кнопку "Войти"

## 📰 Управление новостями

После входа вы сможете:
- Нажимать кнопку **"Управление новостями"** для добавления/редактирования новостей
- Добавлять новые новости с заголовком, содержанием и кратким описанием
- Редактировать существующие новости
- Удалять новости

## ℹ️ Управление разделом "О нас"

- Нажимать кнопку **"О нас"** для редактирования информации о спортивной школе
- Изменять заголовок, подзаголовок и описание
- Редактировать статистику (лет работы, воспитанников, видов спорта)
- Все изменения автоматически сохраняются и отображаются на основном сайте

## 🔧 Если кнопки не работают

1. **Проверьте консоль браузера** (F12 → Console):
   - Должны появляться сообщения при клике на кнопки
   - Если есть ошибки - скопируйте их текст

2. **Убедитесь, что вы вошли в систему:**
   - Должна быть видна административная панель
   - Если перенаправляет на страницу входа - введите логин/пароль еще раз

3. **Проверьте localStorage:**
   - Откройте консоль (F12)
   - Введите: `localStorage.getItem('adminToken')`
   - Должно быть: `"demo-token"`

## 🐛 Поиск проблем

Если кнопки не работают:
1. Откройте консоль разработчика (F12)
2. Нажмите на кнопку "Управление новостями" или "О нас"
3. Посмотрите сообщения в консоли
4. Сообщите об ошибках

## 💾 Сохранение данных

Все данные сохраняются в localStorage браузера:
- Новости: `gorkiNews`
- Информация "О нас": `gorkiAbout`
- Галерея: `gorkiGallery`
- Главный экран: `gorkiHeroImage`
- Цены: `gorkiPrices`
- Контакты: `gorkiContacts`
